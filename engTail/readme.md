- 사람의 수와 단어의 순서가 주어지며 앞의 단어와 연결되지 않을 때 게임이 종료되고 그 번호를 말한 사람이 걸린다, 누가 몇번째에 걸렸는지 반환
- sol1. 앞의 단어의 마지막 스펠링과 뒤에 오는 단어를 비교, 종료되는 지점의 인덱스 값을 

# 입력
    - n : 인원수를 알려주기 위한 변수
    - words : 단어들을 순서대로 나열한 리스트


# 출력
    - 틀린 사람의 번호, 틀린 차례
