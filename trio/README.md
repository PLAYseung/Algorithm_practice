
## [삼총사](https://school.programmers.co.kr/learn/courses/30/lessons/131705)

<br/> 학생 3명의 정수 번호를 더했을 때 0이되면 삼총사이다
<br/> 리스트에 주어진 숫자 3개를 더해서 0이 나오는 경우의 수

## 요소

- 입력 : 정수 리스트
  - [-2, 3, 0, 2, -5]
  - 리스트에 같은 숫자가 존재 할 수 있다

- 출력 : 3 숫자의 합이 0이 되는 경우의 수
  - 2
