# 문자열 압축

## 문제

- 같은 문자 패턴을 압축 했을 때 전체 문자의 길이가 가장 짧게 나오는 패턴의 길이


## 요소

- 입력 : s ( 소문자로만 이루어진 문자열, 길이 1 이상 1000 이하 )

- 출력 : result


## 예상도

1. 문자열 → 문자 하나 하나 리스트 

2. 0번째 문자 1번째 문자 비교
    - true → 문자의 갯수 + 문자
    - flase → 1번째 문자와 2번째 문자 비교
